# Dash-Dost Analytics Dashboard

A robust, interactive analytics dashboard application built with React, Vite, and TypeScript.

## Features
- **Interactive Data Visualization**: Integrated geographic map charts and various data charts for insights, with animated transitions and drill-down highlighting.
- **Dynamic Filtering**: Robust, global filtering engine to synchronize data across all widgets.
- **Deep Exploration**: Drill-down capabilities to zoom in on specific geographic regions and dynamically update widgets based on selections with multi-selection support.
- **Visual Section Containers**: Organize dashboard components into logical visual groups for better structure.
- **Local-First Management**: Refined state management workflows using IndexedDB for high-fidelity session persistence and multi-device syncing safety.
- **Data Import**: Supports CSV/Excel data uploads with automatic parsing and normalization of geographic data (ISO-3166 for accurate mapping).

## Technology Stack
- **Frontend**: React 18+, Tailwind CSS, Motion/React, Recharts, D3.js.
- **Tooling**: Vite, TypeScript, Zustand (state management).
- **Backend (Optional)**: Node.js/Express for API handling and potential integrations.

## Getting Started
1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.
